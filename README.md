# Flappy Bird Clone

Este projeto é um clone do clássico Flappy Bird, desenvolvido como parte de uma aula do curso de Desenvolvedor Fullstack Turbo da Tipscode.
O objetivo principal foi praticar conceitos fundamentais de JavaScript, com foco em manipulação de DOM, lógica de jogo e atualização dinâmica de elementos na tela.

## Sobre o projeto

O jogo replica a mecânica básica do Flappy Bird:
Controle de um personagem que "voa" ao pressionar uma tecla
Obstáculos que se movem continuamente pela tela
Sistema de colisão
Contagem de pontos baseada na progressão do jogador

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (Vanilla)

## Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos importantes como:

- Manipulação de DOM
- Controle de eventos de teclado
- Uso de timers e loops com setInterval
- Lógica de colisão
- Organização de código em JavaScript
- Atualização dinâmica da interface
